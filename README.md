# PulseOps

Real-time dark store operations intelligence dashboard for quick-commerce platforms like Blinkit.

## Tech Stack

- Frontend: React + Redux Toolkit, Tailwind CSS, Recharts, React Router v6
- Backend: Node.js + Express
- Real-time: Socket.io

## Setup

1. Install dependencies: `npm run install-all` or manually in each folder.
2. Run development server: `npm run dev`

This will start both backend on port 3001 and frontend on port 3000.
